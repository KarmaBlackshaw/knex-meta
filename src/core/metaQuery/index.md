### Valid Operators
- `=` Equals _(default)_
- `<` Less than
- `<=` Less than or equal
- `>` Greater than
- `>=` Greater than or equal
- `like` Like
- `between` Between
- `not between` Not Between
- `in` In
  - `value` is expected to be an array
- `not in` Equals
  - `value` is expected to be an array
- `null` Equals
- `not null` Equals
- `not` Equals

### Valid filter fields
- `$and`
- `$or`
- `$not`
